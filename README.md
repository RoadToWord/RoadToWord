# 🛣️ RoadToWord

**RoadToWord** is a high-fidelity, interactive English vocabulary learning platform designed for learners, teachers, and classrooms. Built with a focus on modern design and pedagogical efficiency, it transforms the road to word mastery into an engaging, gamified journey.

---

## 💎 Premium Design: "Sunset Amber & Liquid Glass"
RoadToWord features a state-of-the-art **Cyber-Glass** aesthetic, providing a boutique user experience:
- **Atmospheric Atmosphere:** A breathing, "WebGL-style" backdrop with drifting amber nodes and deep blur effects.
- **True Glassmorphism:** High-spec `backdrop-filter` architecture for all cards, navigation, and interactive elements.
- **Dynamic Interaction:** Liquid sheen animations, neon aura glows on focus, and magnetic scale-up hover effects.
- **Premium Typography:** Clean and modern font stack using **Space Grotesk** and **Inter**.

---

## 🚀 Core Features

### 🎓 For Learners
- **Adaptive Quiz Engine:** Multiple modes including Turkish Meaning, Synonym/Antonym, Definition, Typing, and high-pressure Timed modes.
- **Voice-Enabled Learning:** Native Text-to-Speech (TTS) and Speech Recognition for pronunciation mastery.
- **Gamified Progression:** Streaks, daily goals, and tiered achievements (Bronze to Master).
- **Comprehensive Feedback:** Detailed word breakdowns after every answer, including example sentences and expanded definitions.
- **"Rabb-it" Game:** A floating sidebar pet that grows based on your daily practice.

### 🏫 For Teachers & Admins
- **Unified Dashboard:** Real-time analytics for classrooms and individual learner performance.
- **Classroom Management:** Create classrooms, manage memberships, and assign specific vocabulary packs.
- **Bulk Operations:** CSV import/export for students and vocabulary lists.
- **Audit Systems:** Full audit logs for administrative transparency and user management (Admin/Teacher/Student).

---

## 🛠️ Technology Stack
- **Core:** PHP (Modern 8.x)
- **Database:** SQLite with high-spec PDO optimization.
- **Frontend:** Pure Vanilla CSS (No bloat, high-perf Glassmorphism) & Javascript (no heavy frameworks).
- **Authentication:** Secure session management + **Google OAuth 2.0** integration.
- **Security:** CSRF Protection, mini-WAF for rate limiting, and Cloudflare Turnstile integration.
- **SEO/Ops:** Auto-generated Sitemaps, micro-SEO keyword management, and GeoLite location support.
